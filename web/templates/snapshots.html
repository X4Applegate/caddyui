Config snapshots

Point-in-time copies of Caddy's full live config. Auto-snapshots are taken before every sync (newest 20 kept). Manual snapshots are never pruned. Restoring pushes a snapshot via /load after validation, and auto-snapshots the current state first so the restore itself is undoable.

Import snapshot from file
Upload a JSON file (previously downloaded snapshot, or any Caddy /config export). Stored as a manual snapshot — restore it explicitly once it's in the list.
Auto-snapshots
When enabled, CaddyUI captures the live config before every sync and every restore. Disable to stop pre-sync snapshots — manual snapshots still work either way.
