Which HTTP redirect code should I pick?
  • 301Moved Permanently — browsers and search engines cache this forever. Best for SEO when a URL has permanently changed. (GET/POST may become GET)
  • 302Found / Temporary — not cached. Use when the redirect is temporary or might change. (GET/POST may become GET)
  • 307Temporary Redirect — like 302 but preserves the original method (POST stays POST). Safer for APIs.
  • 308Permanent Redirect — like 301 but preserves the original method. Modern replacement for 301.
Not sure? Pick 301 for permanent domain moves, or 302 for short-term redirects.
