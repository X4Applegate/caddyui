Paste Caddyfile

Paste one or more site blocks. Each block goes through Caddy's /adapt to convert it to JSON, then stored as an Advanced route. The original Caddyfile text is kept alongside the JSON so the UI shows your familiar syntax.

Snippet definitions (name) { ... } and the global options block are skipped automatically. Snippets from the mounted Caddyfile are auto-loaded — so you can import common_security_headers without pasting the definition.
