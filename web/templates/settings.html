Stream Caddy's access logs into CaddyUI for per-host visitor and pageview stats. No cookies, no query strings — only host, path, status, and truncated user-agent are stored. Retained 30 days.

Default {{.AnalyticsTargetPlaceholder}} assumes both containers share a Docker network and CaddyUI's service name is caddyui. On host-network deployments use host.docker.internal:9019 (macOS/Windows) or the LAN IP (Linux). The listener binds on :9019 inside the CaddyUI container — no host port exposed by default.

Drops events from these IPs/CIDRs before the DB insert — useful for hiding your own LAN, an uptime-monitor provider, or a VPN subnet. Changes take effect immediately (no Caddy reload required).
